###**Usage**

####**Clone repo:**

```
git clone https://github.com/d-zukowski/questions.git
```

---

####**Install dependencies:**

```
npm install
```

---

####**Run the project:**

**Build version with local server:**

```
gulp
```

---

If you want optimalized version, generate build and use:

**Dist version:**

```
gulp dist
```

or

**Dist version with local server:**

```
gulp dist:server
```
